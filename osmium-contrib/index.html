Osmium Contrib

Various programs showing what you can do with the Osmium Library.

Programs

In the osmium-contrib repository we collect little programs that use the Osmium Library in various ways as examples what can be done. If you have done something with Osmium and want to share it, this might be a good place. Contact us via info@osmcode.org.

Currently the following programs are available:

  • node_density - Create image showing OSM node density
  • pub_names - Extract names of pubs from OSM file
  • road_length - Calculate length of highways from an OSM file

Code

The code is available from GitHub: github.com/osmcode/osmium-contrib.

Documentation

See the README files for the individual programs for details.
