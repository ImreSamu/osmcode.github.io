Learn about Osmium

Each project on this site has its own documentation. See each project page for details. Here are some good starting points:

FAQ

Some of your questions are probably answered in the FAQ.

Osmium Library Manual

The Osmium Library manual is available in different formats:

  • HTML
  • PDF
  • EPUB
Some of the contents of this manual are also important for developers using Node Osmium or PyOsmium.

Osmium Library Reference

The C++ reference documentation for the Osmium Library is automatically generated using Doxygen. It is, of course, important for C++ developers, but developers using Node Osmium or PyOsmium might want to refer to it, too. It is available for several versions:

  • latest (latest released version)
  • master (development version)

Osmium Tool Manual Pages

The Osmium Tool comes with a set of manual pages available from the project page.

OSM Wiki

Some more information about Osmium can be found in the OSM Wiki. But be aware that this information is mostly about the "Old" Osmium.

OPL File Format Manual

Description of the OPL file format, a text-based format for OSM data, created for use with scripting languages and typical Unix text utilities.
