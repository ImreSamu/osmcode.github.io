Osmium Library

A fast and flexible C++ library for working with OpenStreetMap data.

Note that this is a new beta version of the old Osmium library at github.com/joto/osmium. Feel free to play around with it but be aware that details will change!

Features

The Osmium Library has extensive support for all types of OSM entities: Nodes, Ways, Relations, and Changesets. It allows reading from and writing to OSM files in XML and PBF formats, including change files and full history files. Osmium can store OSM data in memory and on disk in various formats and using various indexes. Its easy to use handler interface allows you to quickly write data filtering and conversion functions. Osmium can ceate WKT, WKB, OGR and other geometries for easy conversion into many GIS formats.
