Detailed Description

template<typename T, template< typename > class TDerived>
class osmium::detail::mmap_vector_base< T, TDerived >

This is a base class for implementing classes that look like STL vector but use mmap internally. This class can not be used on it's own. Use the derived classes mmap_vector_anon or mmap_vector_file.
